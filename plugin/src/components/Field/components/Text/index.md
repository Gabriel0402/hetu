## API

| 参数         | 说明                             | 类型    | 默认值 |
| ------------ | -------------------------------- | ------- | ------ |
| field        | 表单字段                         | string  | -      |
| title        | 字段名称                         | string  | -      |
| defaultValue | 默认值                           | string  | -      |
| jsonFormat   | 格式化展示`json`                 | boolean | false  |
| isWrap       | 是否换行展示(需要后端返回换行符) | boolean | false  |
