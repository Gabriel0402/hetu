## API

| 参数        | 说明                        | 类型   | 默认值 |
| ----------- | --------------------------- | ------ | ------ |
| field       | 表单字段                    | string | -      |
| title       | 字段名称                    | string | -      |
| current     | 指定当前步骤，从 0 开始记数 | number | 0      |
| initial     | 起始序号                    | number | 0      |
| steps       | steps 配置                  | array  | -      |
| title       | 标题                        | string | -      |
| description | 描述                        | string | -      |
| style       | 布局样式                    | object | {}     |
