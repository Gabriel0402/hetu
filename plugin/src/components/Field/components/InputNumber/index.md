## 何时使用

`InputNumber`需要在`Field`内部使用.

## API

### Input

| 参数         | 说明                       | 类型    | 默认值 |
| ------------ | -------------------------- | ------- | ------ |
| defaultValue | 输入框默认内容             | number  |        |
| disabled     | 是否禁用状态，默认为 false | boolean | false  |
| placeholder  | 引导文案                   | string  | -      |
| max          | 最大值                     | number  | -      |
| min          | 最小值                     | number  | -      |
| step         | 每次改变步数，可以为小数   | number  | 1      |
