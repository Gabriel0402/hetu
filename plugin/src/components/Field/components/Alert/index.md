## API

| 参数        | 说明      | 类型    | 默认值 |
| ----------- | --------- | ------- | ------ |
| message     | 提示内容  | string  | -      |
| description | 辅助内容  | string  | -      |
| alertType   | 提示类型  | string  | -      |
| showIcon    | 显示 Icon | boolean | true   |
| colSpan     | 宽度      | number  | -      |
| colPush     | 右移宽度  | number  | -      |
| style       | 布局样式  | object  | -      |
